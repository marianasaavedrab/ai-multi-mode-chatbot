# ai-multi-mode-chatbot
AI chatbot in Python using OpenAI API with memory and dynamic behavior.

## Features

- Conversation memory using message history
- Dynamic system prompts (multiple modes)
- Interactive real-time chat
- Context management (limited history for performance)

## Technologies

- Python
- OpenAI API
- Large Language Models (LLM)
- Prompt Engineering

## Setup
1. Clone the repository:

```bash
git clone https://github.com/your-username/ai-multi-mode-chatbot.git
cd ai-multi-mode-chatbot
```

2. Install dependencies:

```
pip install openai python-dotenv
```
3. Create a .env file in the project directory and add your OpenAI API key:

```
OPENAI_API_KEY=your_api_key
```
